## **ğŸš€ How It Works (Explained)**

This section explains the entire workflow in simple steps:

1. User opens the laptop request form.
2. User fills in all required details.
3. UI Policies adjust the form for correctness.
4. User submits the request.
5. Manager approves or rejects the request.
6. The IT team fulfills the request and updates the status.
